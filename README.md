

包含 注册、登录、最近会话列表、好友列表、聊天 等界面。

业务服务器：包括 注册登录等，用的 Python 和 Mysql
文件服务器：上传下载语音，图片，视频 等，用的 Java Servlet
聊天服务器：收发消息，加删好友，用的 Java 和 Mysql，后又转为 性能更好的 Netty。
